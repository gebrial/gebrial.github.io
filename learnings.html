SQL Basics: select and insert Keywords

20:00, Wednesday, April 29, 2015.

Two very important keywords in the sql database language are select and insert. select allows you to request information from the database and insert allows you to deposit information into the database.

You must first specify the name of the column to withdraw and then the name of the database with the keyword from, e.g., select colm1 from dtbs1. You can select multiple columns and databases, each separated by a comma. Selecting multiple databases requires combining one column from each database using the where keyword and an assignment operator, '='. If there are two columns with the same name in different databases, you must specify which database your desired column comes from with a dot operator, e.g., select dtbs1.colm1, dtbs2.colm1 from dtbs1, dtbs2 where dtbs1.colm3 = dtbs2.colm4.

A database is essentially a table, and a table is no use if you cannot order it. Thus enters the order by keyword. It accepts a column name as it's first parameter and an optional desc parameter to switch the order.

You can also create a new column as you request for information. These are functions that take in a column name as a parameter in brackets. The functions include count, max, min, etc.. On their own these functions act on the entire database and return a single result. However, they can be made to act on a specific column and return a value for each unique item in the column. This is done by also include the group by keyword. It is followed by a column name.

You can also use the where keyword to limit the output to database entries that only have a specific value. This is also done with an assignment operator. Multiple assignments are separated by an and keyword. However, you cannot use this method with newly created columns as where only acts on the original database. You must use the having keyword to act on any new columns that are created. It is used in the same way as where.

Creating multiple pages of databases is also a breeze with sql. We just use the keyword limit in conjunction with offset. limit takes an integer as an argument and only shows that many items per outputted database. offset also takes an integer, n, and skips the first n values of the outputted database.

You must be careful when outputting data after creating new columns as this essentially collapses the database into a new database with one entry for each unique element in the group by column. Requesting information which might have had different values for a single value of the group by column will most likely yield useless results.

This has been a quick, late evening round-up of sql basics. I'll endevour to add some examples later on as practice but if you're unsure of anything: sql basics.

source: Intro to Relational Databases; Lesson 2.

Semantics

11:38, Tuesday, April 28, 2015.

In HTML, certain elements are used that aid in more than making the web page look nice. They help to give meaning to the different parts and texts in the webpage.

One such element is the section element. Everything in the certain block of text that is related can be placed into one section so that it is 'held together', so to speak. Another element is the article. This element is used for smaller blocks of text that can be read outside of the rest of the text and still make sense. These elements help search engines categorize our website. Search engines can drive a lot of traffic to your website so play nice.

There are also pairs of elements that appear to do the same thing but actually have slightly different uses. The first that springs to mind are the italicize elements. There are two versions of this. The first only italicizes the contained block of text. It has the element name of i. It is used only when you want to convey a different tone of voice to the reader, almost as if the text had quotes around it. The second version has the element name of em, as in emphasis. This also italicizes the contained text, but it also has a meaning to it. As the name suggests, it gives a stressed emphasis to the word.

The second pair of such elements are the bold elements. The first version simply bolds the contained text. It is purely for offsetting the text and suggests no overly important value of the contained text. It is given the element name of b. The second version also bolds the text but adds a strong importance to the contained text. As such, it's element name is strong.

source: Shay Howe Tutorials.
